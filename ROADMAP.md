# OpenConvo Roadmap

OpenConvo is open-source first: local-first data, self-hosting, verified free models, and clear bring-your-own-key support.

## Now

- Public landing page and hosted app route.
- Dynamic verified-free OpenRouter model discovery.
- Hosted free mode with a shared daily quota.
- BYOK OpenRouter and Tavily support.
- File-aware chat with PDF/text/code context.
- Projects, prompt snippets, memory, artifacts, import/export, and command palette.
- Privacy and security pages for public deployments.

## Next

- Persistent hosted quota with Redis/KV for production traffic.
- Better visual smoke tests for critical chat flows.
- More robust hosted-mode abuse protection.
- Screenshot assets and short demo video for the README.
- Public documentation for model fallback behavior and rate limits.
- More file parsing coverage and clearer attachment previews.

## Later

- Optional account-based sync for users who want it.
- Team/workspace sharing as an optional hosted feature.
- Pluggable provider support beyond OpenRouter.
- More agent tools with explicit user approval.
- Desktop app packaging.

## Non-Goals For Now

- Mandatory accounts.
- Billing-first product design.
- Storing user conversations on a central server by default.
- Paid model routing without explicit user opt-in.
